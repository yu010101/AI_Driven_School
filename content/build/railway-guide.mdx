# Railwayの使い方

バックエンドのデプロイに便利なRailwayを解説します。

## Railwayとは

- バックエンド向けPaaS
- PostgreSQL等のDB内蔵
- 環境変数管理が簡単
- $5/月のクレジット（無料枠）

## 向いている用途

- Node.jsサーバー
- Python API
- Discord Bot
- データベース

## デプロイ方法

### 1. アカウント作成

[railway.app](https://railway.app)でGitHub連携

### 2. プロジェクト作成

1. New Project
2. Deploy from GitHub repo
3. リポジトリを選択

### 3. 環境変数設定

Variables タブで設定

## データベース追加

1. プロジェクト → Add → Database
2. PostgreSQL/MySQL/Redis等を選択
3. 接続情報が自動で環境変数に

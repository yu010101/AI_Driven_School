## Claude Codeとは？

あなたの**ターミナル（黒い画面）に住むAIアシスタント**です。

話しかけるだけで、メール返信、議事録要約、データ分析、何でもやってくれます。

---step---

## インストールはコピペ1行

```
npm install -g @anthropic-ai/claude-code
```

これだけ。**30秒で終わります。**

:::before-after{before="AI使えない" after="AI使い放題"}

右のターミナルにコマンドをコピペして Enter を押してください。

## 1コマンドでSEO専門家を雇う

:::flow{steps="🔍 /install-skill seo → 🧩 スキル追加 → 🎯 SEO専門AIに変身"}

スキルのインストールはコマンド1行。以降 `/seo` と呼ぶだけで、キーワード密度・メタタグ・構造化データまで自動チェックしてくれます。

---step---

## 人気スキルTOP4

```
claude /install-skill seo
```

| スキル | 用途 |
|--------|------|
| `seo` | 検索順位を上げる |
| `commit` | コミットを自動化 |
| `frontend-design` | UIを美しく |
| `content-creator` | 記事・SNS投稿 |

まずはSEOスキルから体験しましょう。

右のターミナルにコピペして Enter を押してください。

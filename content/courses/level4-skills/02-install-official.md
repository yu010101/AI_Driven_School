## 1コマンドでSEO専門家を雇う

:::flow{steps="🔍 /install-skill seo → 🧩 スキル追加 → 🎯 SEO専門AIに変身"}

スキルのインストールはコマンド1行。以降 `/seo` と呼ぶだけで、キーワード密度・メタタグ・構造化データまで自動チェックしてくれます。

### インストールの流れ（ステップバイステップ）

1. **ターミナルを開く** — Claude Codeが起動できる状態にする
2. **コマンドを実行** — `claude /install-skill seo` と入力してEnter
3. **自動ダウンロード** — `.claude/skills/seo-mastery/SKILL.md` が作成される
4. **確認メッセージ** — 「スキルをインストールしました」と表示されたら完了
5. **即座に利用可能** — 以降 `/seo` と打つだけで起動する

特別な設定は不要です。コマンド1行で、あなたのClaude CodeにSEO専門家が住み着きます。

### インストール前後の違い

**Before（インストール前）：**
```
claude "このページのSEOをチェックして"
→ 一般的なアドバイスが返る（具体性が低い）
```

**After（インストール後）：**
```
claude /seo
→ キーワード密度: 2.1% ✓
→ タイトルタグ: 32文字 ✓
→ メタ説明: 未設定 ✗ 要追加
→ 構造化データ: なし ✗ FAQ推奨
→ 内部リンク: 3本 △ 5本以上推奨
```

インストール後は**数値ベースの具体的な診断結果**が返ります。曖昧なアドバイスではなく、何を直すべきかが一目でわかります。

## 人気スキルTOP4

```
claude /install-skill seo
```

| スキル | 用途 | おすすめ度 |
|--------|------|-----------|
| `seo` | 検索順位を上げる | ★★★ 最優先 |
| `commit` | コミットメッセージ自動生成 | ★★★ 開発者必須 |
| `frontend-design` | UIデザイン品質向上 | ★★☆ フロント担当向け |
| `content-creator` | 記事・SNS投稿生成 | ★★☆ マーケ担当向け |

### 正しくインストールされたか確認する方法

インストール後、以下のコマンドで動作確認できます。

```
claude /seo
```

診断結果の表が表示されれば成功です。「コマンドが見つかりません」と出たら、インストールコマンドを再実行してください。

### スキルをアンインストールしたい場合

不要になったスキルは削除できます。

```
rm -rf .claude/skills/seo-mastery/
```

SKILL.mdファイルを削除するだけです。設定ファイルの変更は不要。追加もシンプルなら、削除もシンプルです。

### おすすめスターターパック

初めてスキルを使うなら、この3つの組み合わせがおすすめです。

1. **`seo`** — 検索エンジンからの流入を増やす（集客の基盤）
2. **`commit`** — コミットメッセージを自動生成（開発効率UP）
3. **`frontend-design`** — UIの品質を底上げ（見た目の印象改善）

この3つがあれば、**集客 → 開発 → デザイン**の基本サイクルがAI化されます。まずはSEOスキルから体験しましょう。

右のターミナルにコピペして Enter を押してください。

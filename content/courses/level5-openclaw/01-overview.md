# OpenClawとは？

**OpenClaw**は、AIエージェントが24時間365日あなたの会社を経営するシステムです。

この塾を運営する合同会社Radineerが**実際に使っている**仕組みを、あなたも構築できます。

:::before-after
before: 社長が全部やる。財務も営業もマーケも。寝る暇なし
after: AIが経営業務を分担。社長は意思決定に集中
:::

## Before / After：社長の1日

| 時間帯 | 導入前（手動） | 導入後（OpenClaw） |
|--------|---------------|-------------------|
| 7:00 | freeeにログインして売上確認 | Slackに財務サマリーが届いている |
| 8:00 | GA4を開いてPV・CVRチェック | マーケレポートも届いている |
| 9:00 | タスク管理ツールで進捗確認 | 遅延アラートだけ確認すればOK |
| 10:00 | Excelでレポート作成 | CEOダッシュボードが自動生成済み |
| 午後 | 会議・メール対応に追われる | **意思決定と戦略立案に集中** |

**削減時間の目安**: 1日あたり約2〜3時間の経営管理業務を自動化。月間で40〜60時間。

## システム全体像

```
[cron: 毎朝8時] → CEO Agent（統合）
                      │
          ┌───────────┼───────────┐
          │           │           │
       CFO Agent   COO Agent   CMO Agent
       (財務)       (運用)       (マーケ)
          │           │           │
       freee API   tasks/YAML   GA4 + GSC
          │           │           │
          └───────────┼───────────┘
                      │
               Slack Bot 投稿
               (#ceo-dashboard)
```

各エージェントは独立した `SKILL.md` を持ち、CEO Agentがそれぞれを呼び出して統合します。

## 前提条件と事前準備

このレベルを進めるには、以下のAPIキー・アカウントが必要です。

| サービス | 用途 | 取得方法 |
|---------|------|---------|
| freee会計 | 財務データ取得 | freee開発者ページでOAuth2アプリを作成 |
| Google Analytics 4 | PV・CVR分析 | GCPコンソールでAnalytics Data API v1を有効化 |
| Google Search Console | 検索順位監視 | GCPコンソールでSearch Console APIを有効化 |
| Slack | 通知・Bot | api.slack.comでApp作成、Bot Token取得 |

**セットアップ所要時間**: 各APIの認証設定に約30分、全体で約2時間。

## 「24時間運用」の実態

OpenClawは「常時起動」ではありません。**cronで定時実行される仕組み**です。

- 毎朝8時にCEOダッシュボードを生成 → Slackに投稿
- 毎日17時に未決済チェック → アラート通知
- 毎週金曜に週次レポート生成
- Slack Botへの呼びかけ時にオンデマンド実行

つまり、Claude Codeが「24時間待機」しているわけではなく、**必要なタイミングで起動して処理し、終了する**設計です。API呼び出しコストも最小限に抑えられます。

## 4つのAIエージェントを設計しよう

CFO（財務）・COO（運用）・CMO（マーケ）・CEO（統合）。この4部門のAIエージェントを設計するところから始めます。

```

```

## 実装のポイント

- **スモールスタートが鉄則**: まずCFOエージェント1つから始めて、動作確認してから次へ進む
- **SKILL.mdがすべての基盤**: 各エージェントの振る舞いはSKILL.mdファイルで定義する。コードではなく「指示書」
- **失敗しても壊れない設計**: エージェントはデータを「読む」だけ。書き込みや決済操作は行わない
- **段階的に育てる**: 最初は簡易版でよい。運用しながらSKILL.mdを改善していく

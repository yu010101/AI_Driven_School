# OpenClawとは？

**OpenClaw**は、AIエージェントが24時間365日あなたの会社を経営するシステムです。

この塾を運営する合同会社Radineerが**実際に使っている**仕組みを、あなたも構築できます。

:::before-after
before: 社長が全部やる。財務も営業もマーケも。寝る暇なし
after: AIが経営業務を分担。社長は意思決定に集中
:::

---step---

## 4つのAIエージェントを設計しよう

CFO（財務）・COO（運用）・CMO（マーケ）・CEO（統合）。この4部門のAIエージェントを設計するところから始めます。

以下のコマンドで、エージェント構成を設計しましょう。

```
claude "経営を自動化するAIエージェント構成を設計して。CFO・COO・CMO・CEOの4部門で"
```

右のターミナルにコピペして Enter を押してください。

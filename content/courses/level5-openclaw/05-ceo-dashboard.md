# CEOダッシュボード

CEOエージェントは、3つのエージェントの報告を**1画面に統合する頭脳**です。これを見るだけで経営状況が分かります。

:::flow
steps:
  - "💰 CFO報告 + 📋 COO報告 + 📈 CMO報告"
  - "🏢 CEOエージェント"
  - "📊 経営ダッシュボード"
:::

---step---

## 全体を1画面で見る

CFOの財務データ、COOの運営状況、CMOのマーケ数値。バラバラだった情報が1つのダッシュボードにまとまります。

```
claude "/ceo ダッシュボードを生成して"
```

これがOpenClawの真価です。**社長が見るべき画面は、これ1つだけ。**

右のターミナルにコピペして Enter を押してください。

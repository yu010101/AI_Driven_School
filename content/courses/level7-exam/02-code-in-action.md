### 追加練習問題

**Q1：CSVファイルを読み込んで売上TOP10を抽出するコマンドは？**

A：`claude "sales.csvを読んで売上TOP10を表形式で出して"` — Claude Codeはファイル名を指示に含めると自動的に読み込みます。CSVでもJSON でもテキストでも対応可能です。出力形式を「表形式で」と指定すると、整った出力が得られます。

**Q2：複数ファイルを同時に処理する場合の書き方は？**

A：`claude "meeting1.txtとmeeting2.txtを読んで、共通の議題をまとめて"` — 複数ファイル名を指示に含めればOKです。Claude Codeは指示文中のファイル名を認識し、それぞれ読み込んで処理します。ディレクトリ全体を指定する場合は `claude "src/フォルダの全ファイルをレビューして"` のように書けます。

**Q3：処理結果をファイルに保存させるには？**

A：`claude "meeting.txtを読んで要約して、summary.mdとして保存して"` — 出力先のファイル名を指示に含めると、Claude Codeが自動的にファイルを作成して保存します。「保存して」「書き出して」「ファイルに出力して」のいずれの表現でも認識されます。

### Claude Code in Action 学習のコツ

- **ファイル名を明示的に含める** — 「このファイル」ではなく具体的なファイル名を書く
- **出力形式を指定する** — 「表形式」「箇条書き」「Markdown」など明示する
- **CLAUDE.mdで前提条件を設定** — 毎回同じ形式で出力させたい場合はCLAUDE.mdに定義

### よくある間違い

- ファイルパスを省略する → Claude Codeがどのファイルか特定できず一般的な回答になる
- 出力形式を指定しない → 毎回異なるフォーマットで出力される
- 存在しないファイルを指定する → エラーメッセージを確認してパスを修正する

## 練習問題

**Q1**: Claude Codeでファイルを読み込ませる際、最も確実な方法はどれか？
A) ファイルをドラッグ＆ドロップする  B) 指示文にファイル名を明記する  C) カレントディレクトリを変更する  D) 環境変数にパスを設定する
→ 正解: B。`claude "report.csvを読んで分析して"` のように、指示文中にファイル名を含めることでClaude Codeが自動的にファイルを読み込みます。

**Q2**: 出力形式を指定する際のベストプラクティスとして正しいのはどれか？
A) 形式は指定しなくてもAIが最適なものを選ぶ  B) 「表形式で」「箇条書きで」「JSON形式で」のように明示する  C) 出力形式はCLAUDE.mdでしか設定できない  D) Markdown形式のみ対応している
→ 正解: B。出力形式を明示することで、毎回一貫したフォーマットの結果が得られます。CLAUDE.mdに定義すれば毎回の指定が不要になります。

**Q3**: CLAUDE.mdに出力ルールを定義するメリットとして正しいのはどれか？
A) 処理速度が向上する  B) 毎回の指示で形式を指定する手間が省ける  C) ファイルサイズが小さくなる  D) エラーが発生しなくなる
→ 正解: B。CLAUDE.mdに「出力は常にMarkdownの表形式で」と書いておけば、個別の指示で毎回指定する必要がなくなります。チーム全体で統一された出力形式を維持する効果もあります。

## 学習のポイント
- ファイル操作の基本は「ファイル名を指示に含める」こと
- 出力形式は「表形式」「箇条書き」「JSON」「CSV」など明示的に指定する
- CLAUDE.mdで出力ルールを事前定義すれば、毎回の指定が不要になる
- 複数ファイルの同時処理も自然言語で指示可能
- 対応レッスン: Level 1「ワンショット実行」、Level 2「スクリプトで仕組み化」

## 実力チェック — 応用問題

**Q4**: 以下のコマンドのうち、最も精度の高い結果が期待できるのはどれか？
A) `claude "このデータを分析して"`
B) `claude "sales_2024.csvを読んで、月別売上を降順で表形式にまとめて"`

D) `claude "データ分析"`
→ 正解: B。ファイル名・処理内容・出力形式の3つを明確に指定しているため、意図通りの結果が返ってきます。曖昧な指示はAIの推測が入り、期待外れの結果になりやすいです。

**Q5**: スクリプトでClaude Codeを定期実行する場合、どのモードを使うべきか？
A) 対話モード  B) ワンショットモード  C) デバッグモード  D) バッチモード
→ 正解: B。ワンショットモード（`claude "指示"`）は標準入出力に対応しており、シェルスクリプト・cron・GitHub Actionsなどの自動化パイプラインに組み込めます。

# コンテンツマーケティング入門

広告費ゼロで集客する最強の方法。

## コンテンツマーケティングとは

```
価値あるコンテンツを作る
  ↓
見込み客が見つける
  ↓
信頼関係が生まれる
  ↓
顧客になる
```

## コンテンツの種類

### ブログ記事

```
・検索流入（SEO）
・長期的に資産になる
・深い内容を伝えられる
```

### SNS投稿

```
・拡散力がある
・即時性がある
・ファンを作りやすい
```

### 動画

```
・理解しやすい
・人柄が伝わる
・YouTubeからの流入
```

### ニュースレター

```
・直接届く
・関係を深められる
・販売につながりやすい
```

## ファネルに合わせたコンテンツ

### 認知（TOFU）

```
目的: 存在を知ってもらう
内容: How-to、入門記事、業界トレンド
例: 「AIコーディングツール10選」
```

### 興味（MOFU）

```
目的: 興味を持ってもらう
内容: 比較記事、ケーススタディ、詳細解説
例: 「Cursor vs Copilot 徹底比較」
```

### 検討（BOFU）

```
目的: 購入を決めてもらう
内容: 導入事例、FAQ、デモ
例: 「○○で開発時間を50%削減した話」
```

## コンテンツカレンダー

### 月間計画

```
週1: ブログ記事
週2: ブログ記事
週3: 動画
週4: まとめ記事

毎日: SNS投稿
```

### テーマのバランス

```
40%: 教育コンテンツ（How-to）
30%: エンタメ/ストーリー
20%: キュレーション
10%: プロモーション
```

## 効果測定

### 見るべき指標

| 指標 | 内容 |
|-----|-----|
| PV | ページビュー |
| UU | ユニークユーザー |
| 滞在時間 | 読まれているか |
| 直帰率 | 興味を持ったか |
| CVR | 登録/購入につながったか |

### コンテンツのROI

```
投資: 記事1本を書く時間（4時間）
リターン: 月間100PV、10登録、1購入

LTV 10,000円なら、1記事で月1万円の価値
```

## 再利用戦略

### 1つのコンテンツを展開

```
ブログ記事
  ↓
・Twitter スレッド
・YouTube 動画
・ニュースレター
・インフォグラフィック
・ポッドキャスト
```

### エバーグリーンコンテンツ

```
時間が経っても価値が下がらない記事：
・入門ガイド
・チェックリスト
・用語解説
・ベストプラクティス
```

## 継続のコツ

### 小さく始める

```
❌ 週3記事を目標
⭕ 週1記事を継続
```

### テンプレート化

```
・構成をパターン化
・タイトルの型を持つ
・リサーチ方法を固定
```

### バッチ処理

```
月初にアイデア出し
週末に執筆
平日に編集・公開
```

# メルマガマーケティング入門

SNSは届くかわからない。メールは確実に届く。

## メルマガのメリット

```
・開封率20-30%（SNSより高い）
・フォロワーを「所有」できる
・長文でも読んでもらえる
・販売への導線が作りやすい
```

## 配信ツール

| サービス | 無料枠 | 特徴 |
|---------|-------|-----|
| Buttondown | 100人 | シンプル、開発者向け |
| ConvertKit | 1,000人 | クリエイター向け |
| Mailchimp | 500人 | 機能豊富 |
| Substack | 無料 | 有料購読に強い |

## 購読者を集める

### フォーム設置場所

```
・LPのファーストビュー
・ブログ記事の末尾
・ポップアップ（控えめに）
・プロフィールページ
```

### 登録の動機を作る

```
❌「ニュースレターに登録」
⭕「毎週、AI開発のTipsをお届け」
⭕「登録で○○ガイドをプレゼント」
```

### リードマグネット

```
登録特典の例：
・PDFガイド
・チェックリスト
・テンプレート
・限定動画
・早期アクセス
```

## コンテンツ設計

### 配信頻度

```
週1回が理想
・少なすぎ → 忘れられる
・多すぎ → 解除される
```

### コンテンツの種類

```
1. 教育系（How-to、Tips）
2. キュレーション（リンク集）
3. ストーリー（体験談、失敗談）
4. ニュース（業界動向）
5. 裏話（開発の舞台裏）
```

### 構成テンプレート

```
件名: 【AI駆動塾】今週のTips: ○○

---

こんにちは、○○です。

今週のトピック：
1. ○○
2. △△
3. □□

【1. ○○について】
...

【お知らせ】
新しい記事を公開しました：
→ [記事タイトル](URL)

---
解除はこちら: [解除リンク]
```

## 件名のコツ

### 開封される件名

```
⭕ 具体的な数字: 「3つの方法」「5分で」
⭕ 疑問形: 「○○してませんか？」
⭕ 限定感: 「今週だけ」「先着」
⭕ 好奇心: 「実は...」「意外な」
```

### 避けるべき件名

```
❌ 煽りすぎ: 「衝撃！」「緊急！」
❌ 曖昧: 「お知らせ」「今週のニュース」
❌ 長すぎ（20文字以内が理想）
```

## 効果測定

### 見るべき指標

| 指標 | 目安 |
|-----|-----|
| 開封率 | 20-30% |
| クリック率 | 2-5% |
| 解除率 | 0.5%以下 |

### 改善ポイント

```
開封率が低い → 件名を改善
クリック率が低い → コンテンツを改善
解除率が高い → 頻度・内容を見直し
```

## 自動化

### ウェルカムメール

```
登録直後:
「ありがとうございます！
まずはこの3つの記事がおすすめです...」
```

### ステップメール

```
Day 1: ウェルカム + 自己紹介
Day 3: 人気記事の紹介
Day 7: 無料リソースの案内
Day 14: 有料プランの紹介
```

# インフルエンサー活用

大企業じゃなくても、インフルエンサーは使える。

## インフルエンサーの分類

| 分類 | フォロワー | 特徴 |
|-----|----------|------|
| メガ | 100万+ | 認知拡大、高コスト |
| マクロ | 10-100万 | バランス型 |
| マイクロ | 1-10万 | エンゲージ高、低コスト |
| ナノ | 〜1万 | 信頼度高、無料〜低コスト |

### 個人開発におすすめ

```
マイクロ〜ナノインフルエンサー
・コストが低い
・ニッチな層にリーチ
・本物の推薦に見える
```

## 探し方

### SNSで検索

```
X（Twitter）:
「#個人開発」「#バイブコーディング」
で発信している人

YouTube:
「プログラミング 初心者」
でチュートリアル出してる人
```

### 条件

```
✓ ターゲットに近いフォロワー
✓ エンゲージメント率が高い
✓ 過去の投稿の質が良い
✓ 広告っぽくない発信
```

### エンゲージメント率の目安

```
(いいね + コメント) / フォロワー数

5%以上: 優秀
2-5%: 良好
2%以下: 低め
```

## 依頼方法

### DMで連絡

```
件名: コラボのご相談（○○より）

○○さん

初めまして。○○を開発している山田と申します。
いつも○○さんの発信を拝見しています。

○○というサービスを運営しており、
もしよろしければ使っていただき、
感想を発信いただけないでしょうか？

【提供できるもの】
・サービスの無料利用
・○○円の報酬（交渉可）

ご興味があれば、詳細をお伝えします。
```

### やり取りのポイント

```
・丁寧に
・相手のメリットを明確に
・押し売りしない
・返事がなくても追わない
```

## 報酬設計

### 報酬の種類

```
1. 無料提供
   サービスを無料で使ってもらう

2. アフィリエイト
   成果報酬型（登録1件○○円）

3. 固定報酬
   1投稿○○円

4. 商品提供
   物理商品を送る
```

### 相場（マイクロ）

```
X投稿: 1-5万円
YouTube: 5-20万円
Instagram: 2-10万円
ブログ: 3-10万円
```

### 無料でやってもらうコツ

```
・本当に良いプロダクトを作る
・相手にもメリットがある関係
・「使ってみて、良かったら紹介して」
```

## 依頼内容

### 明確にすること

```
・投稿の形式（レビュー、チュートリアル等）
・含めてほしい内容
・NG事項
・投稿期限
・リンク/ハッシュタグ
```

### 自由度を残す

```
ガチガチに指定すると広告っぽくなる
→ 「自由に感想を伝えてください」が効果的
```

## 効果測定

### 追跡方法

```
・専用クーポンコード
・UTMパラメータ付きリンク
・「どこで知りましたか？」アンケート
```

### 見るべき指標

```
・リーチ/インプレッション
・エンゲージメント
・サイト流入
・登録/購入数
・CPA（獲得単価）
```

## 注意点

### PR表記

```
広告・PRであることを明示
「#PR」「提供:○○」

ステマは信頼を失う
```

### 相性の確認

```
・過去の投稿を見る
・炎上リスクを確認
・価値観が合うか
```

## 長期的な関係

### アンバサダー化

```
良い関係が築けたら:
・継続的なコラボ
・アンバサダー契約
・共同開発
```

### Win-Winの関係

```
インフルエンサー: コンテンツのネタ、報酬
あなた: 認知拡大、信頼獲得
```

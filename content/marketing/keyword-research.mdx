# キーワードリサーチの方法

効果的なキーワードを見つける方法を解説します。

## 使用ツール（無料）

| ツール | 用途 |
|--------|------|
| ラッコキーワード | 関連KW |
| Google検索 | サジェスト |
| Google Trends | トレンド |
| Search Console | 実績確認 |

## リサーチの流れ

### 1. シードキーワード

自分のテーマに関連する単語

```
例: バイブコーディング、個人開発、AIアプリ
```

### 2. 関連キーワード展開

ラッコキーワードで展開:

```
バイブコーディング
├─ バイブコーディング やり方
├─ バイブコーディング ツール
├─ バイブコーディング 初心者
└─ バイブコーディング Cursor
```

### 3. 検索意図の分類

| 意図 | 例 |
|------|-----|
| 知りたい | 「〜とは」「〜 方法」 |
| やりたい | 「〜 作り方」「〜 手順」 |
| 買いたい | 「〜 おすすめ」「〜 比較」 |

### 4. 競合確認

実際に検索して確認:
- 上位10件の内容
- 記事の質
- 勝てそうか判断

## キーワード選定基準

```
✅ 選ぶべきキーワード:
- 検索意図が明確
- 競合が弱い
- 自分が詳しいテーマ
- CVにつながる

❌ 避けるべきキーワード:
- 競合が強すぎる
- 検索意図が曖昧
- 自分に専門性がない
```

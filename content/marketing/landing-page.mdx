# ランディングページの作り方

ランディングページ（LP）は、**サービスの「顔」**です。

個人開発したサービスを売るために、効果的なLPの作り方をマスターしましょう。

## LPの基本構成

成功するLPには、共通の構成があります。

### 必須セクション

| 順番 | セクション | 役割 |
|------|-----------|------|
| 1 | ヒーロー | 第一印象を決める |
| 2 | 課題提起 | 共感を得る |
| 3 | 解決策 | サービスを紹介 |
| 4 | 機能紹介 | 具体的な価値 |
| 5 | 社会的証明 | 信頼を獲得 |
| 6 | 料金 | 価格を明示 |
| 7 | FAQ | 不安を解消 |
| 8 | CTA | 行動を促す |

## 各セクションの書き方

### 1. ヒーローセクション

**3秒で伝える**のがポイント。

```
構成:
- キャッチコピー（1行）
- サブコピー（2行以内）
- CTAボタン
- ヒーロー画像/動画
```

**良い例:**
```
【キャッチコピー】
AIが自動で議事録を作成

【サブコピー】
会議の録音をアップロードするだけ。
要約・ToDoリストまで自動生成。

【CTA】
無料で試す
```

### 2. 課題提起

ユーザーの**痛み**を言語化します。

```
「こんな経験ありませんか？」

- 会議の議事録を書くのに1時間かかる
- 重要なポイントを聞き逃してしまう
- 決まったことが曖昧になる
```

**コツ**: ターゲットが「そうそう！」と頷く内容を書く。

### 3. 解決策

サービスが**どう解決するか**を説明。

```
「〇〇なら、これらの問題を解決できます」

【Before】議事録作成に1時間
【After】アップロードして5分で完成
```

### 4. 機能紹介

**3〜5つの機能**に絞って紹介。

```
機能1: 自動文字起こし
→ AIが音声を正確にテキスト化

機能2: 要約生成
→ 長い会議も3行でまとめる

機能3: ToDoリスト抽出
→ 決まったことを自動でリストアップ
```

**ポイント**: 機能ではなく「ベネフィット」を書く。

```
✕ 「AI搭載の文字起こし機能」
○ 「手作業ゼロで議事録が完成」
```

### 5. 社会的証明

**信頼性**を高めます。

```
活用できるもの:
- ユーザーの声（実名・顔写真があるとベスト）
- 導入企業ロゴ
- 数字（「1,000社が利用」など）
- メディア掲載実績
```

**個人開発の場合:**
```
- X（Twitter）の感想ツイートを引用
- 友人・知人に使ってもらってフィードバックを掲載
- ベータユーザーの声を載せる
```

### 6. 料金セクション

**シンプルに**伝えます。

```
プラン構成:
- 無料プラン: 機能を限定
- 有料プラン: フル機能
- （必要なら）エンタープライズ

料金表示のコツ:
- 一番人気のプランを目立たせる
- 年払いの割引を強調
- 「1日あたり○○円」で安く見せる
```

### 7. FAQ

**よくある質問**で不安を解消。

```
Q. 解約はいつでもできますか？
A. はい、いつでもワンクリックで解約できます。

Q. データは安全ですか？
A. 暗号化して保存、第三者には共有しません。

Q. サポートはありますか？
A. メールサポートは24時間以内に返信します。
```

### 8. CTA（最終アクション）

**背中を押す**一言を添えて。

```
今すぐ無料で始める
↓
「14日間無料。クレジットカード不要。」
```

## コピーライティングのコツ

### 1. ターゲットを1人に絞る

```
✕ 「誰でも使える便利ツール」
○ 「毎週の議事録作成に疲れた、あなたへ」
```

### 2. 具体的な数字を使う

```
✕ 「時間を大幅に短縮」
○ 「作業時間を80%カット」
```

### 3. 行動を明確に

```
✕ 「詳しく見る」
○ 「無料で試す」「今すぐ始める」
```

## デザインのポイント

### 余白を活かす

```
詰め込みすぎない
→ 各セクションに十分な余白を
```

### 色は3色まで

```
- メインカラー: 1色
- アクセント: 1色
- テキスト: 黒/白
```

### CTAボタンは目立たせる

```
- アクセントカラーを使う
- サイズは大きめ
- 複数箇所に配置
```

## AIにLPを作ってもらう

### プロンプト例

```
以下のサービスのランディングページを作成して。

【サービス名】
MeetingAI

【概要】
会議の録音から自動で議事録を作成するサービス

【ターゲット】
週に3回以上会議がある会社員

【機能】
- 自動文字起こし
- 要約生成
- ToDoリスト抽出

【料金】
月額980円

【構成】
1. ヒーロー
2. 課題提起
3. 機能紹介
4. 料金
5. FAQ
6. CTA

Tailwind CSSでモダンなデザインにして。
```

## まとめ

LP作成のポイント：

1. **基本構成**を押さえる
2. **ベネフィット**を伝える
3. **社会的証明**で信頼を獲得
4. **CTA**は明確に

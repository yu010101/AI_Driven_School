# プレスリリースの書き方

メディアに取り上げてもらえれば、一気に認知度UP。

## プレスリリースを出すタイミング

```
・新サービスのローンチ
・大型アップデート
・資金調達
・ユーザー数達成
・受賞・認定
・提携発表
```

## 構成

### 基本構成

```
1. タイトル（最重要）
2. サブタイトル
3. リード文（要約）
4. 本文
5. 会社概要
6. 問い合わせ先
```

### タイトルのコツ

```
⭕「AIでコードを書く時代に。開発効率を3倍にする○○をリリース」
⭕「月間1万ユーザー突破。個人開発者向け○○が急成長」

❌「株式会社○○、新サービス開始のお知らせ」
→ 誰も興味を持たない
```

### リード文

```
5W1Hを30秒で読める長さで:
・誰が（Who）
・何を（What）
・いつ（When）
・どこで（Where）
・なぜ（Why）
・どのように（How）
```

## 書き方のコツ

### ニュース価値を明確に

```
記者が「これは読者に伝える価値がある」と思うか？

価値がある:
・初めて
・最大/最小
・トレンドに関連
・社会問題を解決
・数字で示せる成果
```

### 具体的な数字を入れる

```
❌「多くのユーザーに支持されています」
⭕「リリース3ヶ月で10,000ユーザーを突破」

❌「大幅に改善しました」
⭕「作業時間を平均60%削減」
```

### 引用を入れる

```
「開発者として、毎日の作業に追われる中で
このツールのおかげで創造的な仕事に
時間を使えるようになりました」
- 導入企業A社 開発部長 山田氏
```

## テンプレート

```
【タイトル】
○○、AIで開発効率を3倍にする「△△」を正式リリース

【サブタイトル】
〜非エンジニアでもアプリが作れる時代へ〜

【リード】
○○（本社：東京都、代表：山田太郎）は、
2024年12月20日、AIを活用した開発支援ツール
「△△」を正式リリースしました。
本サービスは、コーディング経験がない方でも
自然言語でアプリを開発できるツールです。

【本文】
■ サービス概要
...

■ 開発背景
...

■ 主な機能
...

■ 今後の展望
...

【会社概要】
会社名：○○
所在地：東京都○○区
代表者：山田太郎
設立：2024年1月
事業内容：...
URL：https://example.com

【本件に関するお問い合わせ】
○○ 広報担当
Email: press@example.com
```

## 配信方法

### 無料

```
・PR TIMES（無料プランあり）
・直接メディアに送る
・SNSで発信
```

### 有料

```
・PR TIMES
・@Press
・共同通信PRワイヤー
```

## メディアへの直接アプローチ

### ターゲットメディアをリストアップ

```
・業界メディア
・テックメディア
・地域メディア
・影響力のある個人
```

### ピッチメール

```
件名: 【取材のご相談】AIで開発効率3倍の新ツール

○○編集部 ご担当者様

突然のご連絡失礼いたします。
○○の山田と申します。

先日リリースした「△△」について、
貴媒体でのご紹介をご検討いただけないでしょうか。

【ポイント】
・リリース1週間で○○ユーザー登録
・非エンジニアでもアプリが作れる
・○○業界で注目

ご興味があれば、詳細資料をお送りいたします。

山田太郎
```

## フォローアップ

### 掲載されたら

```
・お礼メール
・SNSでシェア
・サイトに「メディア掲載」として表示
```

### 掲載されなくても

```
・関係を維持
・次のネタがあれば再度連絡
・無理に追わない
```

# コールドメールの書き方

知らない相手に送るメールで、返信をもらうのは難しい。
でも、コツを押さえれば返信率は上がる。

## コールドメールの基本

### 返信率の目安

- 一般的: 1-5%
- 良い: 5-10%
- 優秀: 10%以上

### 成功の条件

1. 相手にとって価値がある
2. パーソナライズされている
3. 行動が明確

## 件名のコツ

### 開封される件名

```
⭕「○○について質問」（短くシンプル）
⭕「△△さんからのご紹介」（紹介は強い）
⭕「〇〇の件でご相談」（具体的）
```

### 開封されない件名

```
❌「素晴らしいご提案があります」
❌「【重要】ご確認ください」
❌「弊社サービスのご案内」
```

### 件名のポイント

- 5-7語以内
- 具体的に
- 煽らない

## 本文の構成

### 1行目: フック

```
「先日の○○のイベントで、御社の取り組みを拝見しました」
「△△さんのXで、○○についてお悩みと拝見しました」
```

**ポイント**: なぜ連絡したか、相手に関する具体的な言及

### 2-3行目: 価値提案

```
「私たちは○○というツールを提供しており、
△△の課題を□□で解決しています」
```

**ポイント**: 相手の課題 + 解決策を1-2文で

### 4行目: 実績（社会的証明）

```
「すでに○○社など100社以上にご利用いただいています」
```

### 5行目: CTA（行動喚起）

```
「来週15分ほどお時間いただけませんか？」
「ご興味があれば、簡単なデモをご用意できます」
```

**ポイント**: 具体的で小さなお願い

## テンプレート例

### 基本形

```
件名: ○○について質問

△△様

○○のブログ記事を拝見し、ご連絡しました。

□□についてお悩みとのこと、私たちの○○が
お役に立てるかもしれません。

すでに△△社など50社以上にご利用いただいており、
平均30%の工数削減を実現しています。

来週、15分ほどオンラインでお話しできませんか？
ご都合の良い日時を2-3つ教えていただければ幸いです。

よろしくお願いいたします。

○○
```

### 紹介経由

```
件名: △△さんからのご紹介

○○様

△△さんからご紹介いただき、ご連絡しました。

御社で□□にお困りと伺い、私たちの○○が
お力になれるかもしれないと思いました。

...
```

### リサーチ結果を活用

```
件名: 御社の○○について

△△様

御社の○○を拝見しました。
特に□□の取り組みが素晴らしいと感じました。

一方で、△△については課題を感じていませんか？

私たちの○○は、まさにその課題を解決するために
開発しました。

...
```

## やってはいけないこと

### 1. 長すぎるメール

```
❌ 10行以上の長文
→ スマホで読まれることを想定する
```

### 2. 自分の話ばかり

```
❌「弊社は2020年に設立され...」
→ 相手は自社の説明に興味がない
```

### 3. 曖昧なCTA

```
❌「ご検討ください」
→ 何をすればいいかわからない
```

### 4. 一斉送信感

```
❌「各位」「関係者様」
→ パーソナライズされていない
```

## フォローアップ

### タイミング

- 1通目: 初回送信
- 2通目: 3日後
- 3通目: 1週間後
- 4通目: 2週間後（最後）

### フォローアップ例

```
件名: Re: ○○について質問

△△様

先日お送りしたメール、ご確認いただけましたでしょうか？

改めて、○○についてご説明させていただければと思います。
今週であれば、水曜か金曜の午後が空いております。

お忙しいところ恐れ入りますが、
ご検討いただけますと幸いです。

○○
```

## 効果測定

### 追跡すべき指標

| 指標 | 目標 |
|-----|-----|
| 開封率 | 40%以上 |
| 返信率 | 5%以上 |
| ミーティング率 | 2%以上 |

### 改善ポイント

- 開封率が低い → 件名を改善
- 返信率が低い → 本文を改善
- ミーティング率が低い → CTAを改善

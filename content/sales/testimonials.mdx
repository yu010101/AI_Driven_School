# 推薦文・レビューの集め方

「この人も使ってるなら安心」は最強の販促。

## 推薦文の効果

### 数字で見る効果

- 購入前にレビューを見る: 93%
- レビューがあると購入率: +270%
- 星4以上で信頼度: 大幅UP

### なぜ効果があるか

1. **社会的証明**: 他の人も使っている安心感
2. **第三者の声**: 自社の宣伝より信頼される
3. **具体的なイメージ**: 使用シーンが想像できる

## 集めるベストタイミング

### 1. 成功体験の直後

```
タスク完了時:
「目標達成おめでとうございます！
よろしければ、感想を聞かせてください」
```

### 2. ポジティブなフィードバック時

```
「素晴らしいです！」と言われたら:
「ありがとうございます！
よろしければ、推薦文としてサイトに掲載させていただけませんか？」
```

### 3. 契約更新時

```
「いつもご利用ありがとうございます。
今後の参考に、ご感想をお聞かせください」
```

### 4. NPS調査で高評価時

```
NPS 9-10の回答者に:
「高評価ありがとうございます！
詳しいご感想を聞かせていただけませんか？」
```

## 依頼の仕方

### メールテンプレート

```
件名: ご感想をお聞かせください

○○様

いつもご利用ありがとうございます。

○○様のような方のご感想は、
これから利用を検討される方の参考になります。

2-3分で回答いただける簡単な質問にお答えいただけませんか？

[回答する]

お忙しいところ恐れ入りますが、
ご協力いただけますと幸いです。
```

### 聞くべき質問

1. **導入前の課題**
   - 「導入前、どんなことに困っていましたか？」

2. **選んだ理由**
   - 「なぜこのサービスを選びましたか？」

3. **得られた成果**
   - 「導入後、どんな変化がありましたか？」

4. **おすすめポイント**
   - 「他の人にすすめるとしたら、どんな点ですか？」

## 良い推薦文の要素

### 具体性

```
❌「とても便利です」
⭕「毎日の作業が30分から5分に短縮されました」
```

### ビフォー/アフター

```
「以前は○○で困っていましたが、
導入後は△△になりました」
```

### 数字

```
「作業時間が70%削減」
「月5万円のコスト削減」
「チーム全員が使いこなせた」
```

### 感情

```
「もう手放せません」
「もっと早く知りたかった」
「チームの雰囲気が変わった」
```

## 推薦文の形式

### テキスト

```
「○○のおかげで、毎日の業務が格段に楽になりました。
特に△△機能は手放せません。導入を迷っている方には
ぜひおすすめしたいです」

- 山田太郎, ABC株式会社 マーケティング部
```

### 動画

```
・1-2分の短い動画
・自然な環境（オフィス、自宅）
・台本なしの自然な話し方
・顔出しで信頼性UP
```

### ケーススタディ

```
【導入企業】ABC株式会社
【課題】月次レポート作成に1週間かかっていた
【解決策】○○の自動レポート機能を導入
【成果】作成時間を90%削減、月5万円のコスト削減
```

## 掲載する場所

### 効果的な配置

1. **トップページ**
   - ファーストビューの下
   - 3-5個をローテーション

2. **料金ページ**
   - 「この価格で得られる価値」の証明

3. **LP（ランディングページ）**
   - CTA近くに配置

4. **SNS**
   - 引用RTでシェア
   - ハイライトに固定

## 謝礼について

### 謝礼を渡すべきか

```
基本的には不要。
「いつもありがとうございます」の気持ちで十分。
```

### 渡す場合

- ノベルティグッズ
- 有料機能の無料提供
- 次回更新時の割引

### 注意点

```
❌ 金銭を渡して良いレビューを書かせる
→ ステマになる可能性

⭕ 正直な感想をお願いし、結果として良い評価
→ 問題なし
```

## 許可の取り方

### 確認すべきこと

```
□ 名前の掲載OK？
□ 会社名の掲載OK？
□ 顔写真の掲載OK？
□ SNSでのシェアOK？
```

### 許可テンプレート

```
「いただいたご感想を、弊社のウェブサイトや
SNSで紹介させていただいてもよろしいでしょうか？

□ 名前と会社名を掲載してOK
□ 名前のみ掲載OK（会社名は伏せる）
□ 匿名で掲載OK
□ 掲載NG
```

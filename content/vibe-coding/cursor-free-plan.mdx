# Cursorの無料プランでどこまでできる？

Cursorには無料プラン（Hobby）があります。「本当に無料で使えるの？」「制限は？」という疑問に答えます。

## 無料プランの内容

| 項目 | 内容 |
|------|------|
| 料金 | $0 |
| Tab補完 | 月2000回 |
| スローリクエスト | 月50回 |
| チャット | 利用可能 |
| Composer | 利用可能 |

## 月2000回で何ができる？

### 1日あたりの計算

- 月2000回 ÷ 30日 = **約66回/日**

### 実際の使用例

```
1回のコーディングセッション（2時間）:
- Tab補完: 30-50回
- チャット: 5-10回
- 合計: 約50回

→ 1日1-2セッションなら余裕
```

## 無料プランでできること

### ✅ できる

1. **簡単なアプリ開発**
   - ToDoアプリ
   - ポートフォリオサイト
   - ランディングページ

2. **学習・練習**
   - コードの書き方を学ぶ
   - AIコーディングに慣れる

3. **週末開発**
   - 趣味プロジェクト
   - 副業の小規模案件

### ❌ 厳しい

1. **毎日のフルタイム開発**
   - 1日8時間使うと枯渇

2. **大規模プロジェクト**
   - ファイル数が多いと消費が激しい

3. **複数プロジェクトの並行**

## 無料枠を最大限活用するコツ

### 1. Tab補完を節約

```
❌ 1文字ごとにTab確認
✅ ある程度書いてからTab確認
```

### 2. チャットで一度に多く質問

```
❌ 「変数名どうする？」「型は？」「エラーハンドリングは？」（3回消費）
✅ 「変数名、型、エラーハンドリングを含めてログイン機能を作って」（1回消費）
```

### 3. Composerを活用

複数ファイルを一度に生成すれば、個別に作るより効率的。

### 4. 月末は節約モード

残り枠が少なくなったら、重要な作業に絞る。

## 使用量の確認方法

1. Cursorを開く
2. 右下のステータスバーを確認
3. または Settings → Usage で詳細確認

## 無料 vs Pro の判断基準

### 無料で十分な人

- 週末だけ開発
- 学習目的
- 小規模プロジェクト
- お試し中

### Proにすべき人

- 毎日開発する
- 仕事で使う
- 複数プロジェクト
- ストレスなく使いたい

## Proプランの価値

| 項目 | Hobby | Pro |
|------|-------|-----|
| 料金 | $0 | $20/月 |
| Tab補完 | 2000回/月 | 無制限 |
| 高速リクエスト | 50回/月 | 500回/月 |
| 体験 | 制限あり | ストレスフリー |

**$20/月 = 約3000円/月**

コーヒー1杯/日より安い。仕事で使うならすぐ元が取れます。

## 無料から始めるロードマップ

```
Week 1-2: 無料プランでお試し
↓
Week 3-4: 使用頻度を確認
↓
判断: 月1000回以下 → 無料継続
      月1000回以上 → Pro検討
```

## よくある質問

### Q. 無料枠を使い切ったらどうなる？

翌月まで待つか、Proにアップグレード。エディタとしての機能は使えます。

### Q. 学生割引はある？

Cursorには学生プランはありませんが、無料プランが充実しているので問題なし。

### Q. いつProにすべき？

「もっと使いたい」とストレスを感じたら。無理に我慢する必要はありません。

## まとめ

- 無料プランは**月2000回のAI補完**
- **週末開発や学習には十分**
- **毎日使うならProを検討**
- まずは無料で始めて、必要に応じてアップグレード

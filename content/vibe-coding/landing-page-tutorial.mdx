# ランディングページの作り方

売上・登録に直結するLPを作ります。

## LPの基本構成

1. **ヒーロー**: キャッチコピー + CTA
2. **課題提起**: ユーザーの悩み
3. **解決策**: あなたの提供価値
4. **特徴**: 3-5つのベネフィット
5. **社会的証明**: 実績、お客様の声
6. **料金**: 明確な価格
7. **FAQ**: よくある質問
8. **CTA**: 最終アクション

## ステップ1: ヒーローセクション

```
[サービス名]のLPのヒーローセクションを作って。

内容:
- キャッチコピー: [ユーザーのベネフィット]
- サブコピー: [補足説明]
- CTAボタン: [アクション名]
- ビジュアル: 右側に画像/モックアップ

デザイン:
- 背景にグラデーション
- 大きなタイポグラフィ
- CTAは目立つ色
```

## ステップ2: 特徴セクション

```
特徴セクションを3つのカードで作って:

特徴1: [タイトル] - [説明]
特徴2: [タイトル] - [説明]
特徴3: [タイトル] - [説明]

デザイン:
- アイコン付き
- ホバーで浮き上がる
- 各カードに異なる色のアクセント
```

## ステップ3: お客様の声

```
お客様の声セクションを作って:
- 3件の testimonial
- 顔写真（丸型）
- 名前、肩書き
- コメント
- 星評価

レイアウト: 横並び、レスポンシブで縦に
```

## ステップ4: 料金セクション

```
料金プランセクションを作って:
- 3プラン（Basic, Pro, Enterprise）
- 各プランの機能一覧
- 推奨プランにバッジ
- CTAボタン

プラン内容:
[各プランの詳細を記載]
```

## ステップ5: FAQ

```
FAQセクションを作って:
- アコーディオン形式
- 5-7個の質問
- クリックで回答表示
- スムーズなアニメーション
```

## ステップ6: 最終CTA

```
ページ最下部にCTAセクションを作って:
- 背景色を変えて目立たせる
- 「今すぐ始める」的なコピー
- 大きなCTAボタン
- 安心感を与える一言（30日間返金保証等）
```

## CVを上げるコツ

- CTAボタンは**目立つ色**
- ボタン文言は**具体的に**（「詳しく見る」より「無料で始める」）
- **スクロールでCTA**が見えるように
- モバイルでの**タップしやすさ**

## A/Bテストすべき項目

- キャッチコピー
- CTAボタンの色・文言
- ヒーロー画像
- 料金の見せ方

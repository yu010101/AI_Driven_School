# Lovable入門ガイド

**Lovable**（旧GPT Engineer）は、プロンプトからフルスタックアプリを生成するAIツールです。

## Lovableとは

| 項目 | 内容 |
|------|------|
| 旧名 | GPT Engineer |
| タイプ | ブラウザベース |
| 料金 | 無料〜$25/月 |
| 出力 | React + TypeScript |

## 始め方

1. [lovable.dev](https://lovable.dev)にアクセス
2. アカウント作成
3. プロンプトを入力
4. アプリが生成される

## 基本的な使い方

```
プロンプト例:
- 「タスク管理アプリ」
- 「ポートフォリオサイト」
- 「ECサイトのカート機能」
```

## 特徴

- **リアルタイムプレビュー**: 生成中にプレビュー確認
- **コード出力**: ダウンロード可能
- **Supabase連携**: データベース対応
- **反復改善**: 追加指示で修正

## 料金

| プラン | 料金 |
|--------|------|
| Free | $0 |
| Pro | $25/月 |

## Bolt.newとの違い

| 項目 | Lovable | Bolt.new |
|------|---------|----------|
| UI | よりモダン | シンプル |
| 連携 | Supabase | Netlify |
| 料金 | やや高め | 同程度 |

## まとめ

- プロンプトからアプリ生成
- リアルタイムプレビューが便利
- Supabase連携でDB対応

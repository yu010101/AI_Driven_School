# Replit Agent入門

**Replit Agent**は、ブラウザで動作するAI開発環境です。自然言語でアプリを作成できます。

## Replit Agentとは

| 項目 | 内容 |
|------|------|
| タイプ | ブラウザベースIDE |
| 料金 | 無料〜$25/月 |
| 特徴 | チーム共同開発対応 |
| 対応言語 | 多数（Python, JS, etc.） |

## 始め方

1. [replit.com](https://replit.com)にアクセス
2. アカウント作成
3. 「Create Repl」→「Agent」を選択
4. プロンプトを入力

## 基本的な使い方

```
指示例:
- 「Pythonでウェブスクレイピングツールを作って」
- 「Flaskでシンプルなブログを作って」
- 「データ可視化ダッシュボードを作って」
```

## 料金

| プラン | 料金 | 内容 |
|--------|------|------|
| Free | $0 | 基本機能 |
| Replit Core | $25/月 | Agent無制限 |

## 特徴

- **マルチ言語対応**: Python, Node.js, Go等
- **チーム開発**: リアルタイム共同編集
- **デプロイ内蔵**: ワンクリックで公開
- **データベース対応**: PostgreSQL等

## 他ツールとの比較

| 項目 | Replit | Bolt.new |
|------|--------|----------|
| 言語 | 多数 | JS中心 |
| チーム | 対応 | 非対応 |
| 料金 | やや高め | 同程度 |

## まとめ

- ブラウザで完結する開発環境
- マルチ言語対応が強み
- チーム開発に便利

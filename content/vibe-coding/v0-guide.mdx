# Vercel v0入門ガイド

**v0**は、Vercelが提供するAI UIジェネレーターです。プロンプトを入力するだけで、美しいReactコンポーネントが生成されます。

## v0とは

### 特徴

| 項目 | 内容 |
|------|------|
| 開発元 | Vercel |
| 特化分野 | UI/コンポーネント |
| 技術 | React, shadcn/ui, Tailwind |
| 料金 | 無料〜$20/月 |
| 出力 | コピペ可能なコード |

### 他ツールとの違い

| ツール | 特徴 |
|--------|------|
| v0 | UI特化、高品質デザイン |
| Bolt.new | フルスタック、デプロイまで |
| Cursor | 汎用エディタ |

## 始め方

### ステップ1: アクセス

1. [v0.dev](https://v0.dev)にアクセス
2. Vercelアカウントでログイン

### ステップ2: プロンプト入力

```
例: 「モダンなダッシュボードのサイドバーを作って」
```

### ステップ3: 生成を待つ

- 数秒〜数十秒で生成
- 複数のバリエーションが提示

### ステップ4: コードをコピー

- 気に入ったデザインを選択
- 「Copy Code」でコピー
- プロジェクトに貼り付け

## 基本的な使い方

### 1. コンポーネント生成

```
良いプロンプト例:
- 「ダークモード対応のナビゲーションバー」
- 「カード型の料金表、3プラン」
- 「モーダルダイアログ、フォーム付き」
```

### 2. 修正・調整

```
追加指示例:
- 「もっとシンプルにして」
- 「色を青ベースに変えて」
- 「アニメーションを追加して」
```

### 3. コードの利用

生成されるコード:
- React + TypeScript
- Tailwind CSS
- shadcn/uiコンポーネント

## プロンプトのコツ

### 良いプロンプト

```
✅ 「ダッシュボードの統計カード。
    売上、ユーザー数、コンバージョン率を表示。
    アイコン付き、上昇/下降の矢印も」
```

### 改善が必要なプロンプト

```
❌ 「かっこいいカード」
→ 具体的に何を表示するか書く
```

## 他ツールとの連携

### Cursor + v0 の組み合わせ

1. **v0**でUIを生成
2. コードをコピー
3. **Cursor**に貼り付け
4. Cursorで機能を追加

### Next.js + v0

v0のコードはNext.jsと相性抜群:

```bash
# shadcn/uiを初期化
npx shadcn-ui@latest init

# v0のコードを貼り付け
```

## よくある質問

### Q. 商用利用できる？

はい、問題ありません。

### Q. 生成されたコードの著作権は？

あなたのものです。自由に使えます。

### Q. 日本語プロンプトは？

使えますが、英語の方が精度が高い場合も。

### Q. フルページは作れる？

コンポーネント単位が得意。ページ全体はBolt.newがおすすめ。

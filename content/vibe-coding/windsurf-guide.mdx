# Windsurf入門ガイド

**Windsurf**は、Codeium社が開発したAIコードエディタです。Cursorの強力な代替として注目されています。

## Windsurfとは

### 基本情報

| 項目 | 内容 |
|------|------|
| 開発元 | Codeium |
| タイプ | デスクトップエディタ |
| ベース | VSCode |
| 料金 | 無料〜$15/月 |
| 特徴 | Cascade（自律的AI） |

### Cursorとの違い

| 項目 | Windsurf | Cursor |
|------|----------|--------|
| 料金 | 無料〜$15 | 無料〜$20 |
| 独自機能 | Cascade | Composer |
| AI | Codeium AI | Claude/GPT |
| 無料枠 | より充実 | 月2000回 |

## インストール方法

### ステップ1: ダウンロード

1. [windsurf.com](https://windsurf.com)にアクセス
2. 「Download」をクリック
3. OS（Mac/Windows/Linux）を選択

### ステップ2: インストール

**Mac:**
```
1. .dmgファイルを開く
2. Applicationフォルダにドラッグ
```

**Windows:**
```
1. .exeファイルを実行
2. インストーラーの指示に従う
```

### ステップ3: 初期設定

1. Windsurfを起動
2. アカウント作成（無料）
3. VSCode設定のインポート（任意）

## 基本的な使い方

### 1. AIチャット

```
ショートカット: Cmd/Ctrl + L

使い方:
1. チャットパネルが開く
2. 質問や指示を入力
3. AIが回答・コード生成
```

### 2. Cascade（自律的AI）

Windsurfの目玉機能。AIが自律的にコードを書きます。

```
使い方:
1. チャットで「Cascade」モードを選択
2. 指示を入力（例：「認証機能を追加して」）
3. AIが自動でファイル作成・編集
4. 途中でレビュー・修正可能
```

### 3. Tab補完

```
1. コードを書き始める
2. 補完候補が表示
3. Tabで採用
```

### 4. インライン編集

```
ショートカット: Cmd/Ctrl + K

1. コードを選択
2. 指示を入力
3. AIが編集
```

## Cascadeの活用例

### 例1: 新機能追加

```
指示: 「ユーザープロフィールページを作って」

Cascadeの動作:
1. 必要なファイルを分析
2. コンポーネント作成
3. ルーティング設定
4. スタイル追加
```

### 例2: リファクタリング

```
指示: 「このコンポーネントをTypeScriptに変換して」

Cascadeの動作:
1. 型定義を推論
2. コードを変換
3. 関連ファイルも更新
```

## よくある質問

### Q. VSCodeの拡張機能は使える？

はい、ほとんどの拡張機能が使えます。

### Q. Cursorから移行できる？

はい、設定をインポート可能です。

### Q. 日本語対応は？

AIは日本語を理解します。UIは英語のみ。

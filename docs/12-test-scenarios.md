# テストシナリオ定義

## 機能一覧

### 1. トップページ（/）
- 正常系: ページが表示される、CTAが表示される、ナビゲーションが動作する
- 異常系: データ取得失敗時のエラー表示

### 2. Aboutページ（/about）
- 正常系: プロフィール情報が表示される
- 異常系: データ取得失敗時のエラー表示

### 3. 無料教材一覧（/free）
- 正常系: 2つの教材が表示される、リンクが動作する
- 異常系: 教材が見つからない場合の表示

### 4. バイブコーディングの教科書LP（/free/vibe-coding-book）
- 正常系: LPが表示される、フォームが動作する（存在する場合）
- 異常系: ページが見つからない場合の404

### 5. 0円マーケティングの教科書LP（/free/zero-marketing-book）
- 正常系: LPが表示される、フォームが動作する（存在する場合）
- 異常系: ページが見つからない場合の404

### 6. 知識DB一覧（/knowledge）
- 正常系: 3つのカテゴリが表示される、リンクが動作する
- 異常系: カテゴリが存在しない場合の表示

### 7. バイブコーディングDB一覧（/knowledge/vibe-coding）
- 正常系: 記事一覧が表示される、ページネーションが動作する
- 異常系: 記事が0件の場合の表示、ページネーションエラー

### 8. 実装DB一覧（/knowledge/build）
- 正常系: 記事一覧が表示される、フィルタリングが動作する
- 異常系: 記事が0件の場合の表示

### 9. マーケDB一覧（/knowledge/marketing）
- 正常系: 記事一覧が表示される、ソートが動作する
- 異常系: 記事が0件の場合の表示

### 10. 記事詳細ページ（/knowledge/[category]/[slug]）
- 正常系: 記事が表示される、内部リンクが動作する、CTAが表示される
- 異常系: 記事が見つからない場合の404、カテゴリが無効な場合の404

### 11. 内部リンク機能
- 正常系: リンクが正しく遷移する、アンカーテキストが適切
- 異常系: リンク先が存在しない場合の404

### 12. CTA機能
- 正常系: 正しいCTAが表示される、リンクが動作する、1ページに1つだけ
- 異常系: CTAが2つ以上表示される、リンクが無効

### 13. SEO機能
- 正常系: sitemap.xmlが生成される、robots.txtが生成される、メタデータが設定される
- 異常系: sitemap生成エラー、メタデータ欠損

### 14. 404エラーページ
- 正常系: カスタム404が表示される、ホームへのリンクが動作する
- 異常系: デフォルト404が表示される

### 15. 500エラーページ
- 正常系: カスタム500が表示される、エラーログが記録される
- 異常系: エラーが適切に処理されない

## ビジネスロジックテスト

### CTA配置ルール
- バイブコーディングDB記事 → バイブコーディングの教科書のみ
- 実装DB記事 → バイブコーディングの教科書のみ
- マーケDB記事 → 0円マーケティングの教科書のみ
- 1ページにCTAは1つだけ

### 内部リンクルール
- 同カテゴリ内リンク: 最低3本
- 別カテゴリリンク: 最低2本
- 合計: 最低5本

### 記事必須要素
- 結論（30秒要約）が存在する
- コピペ可能なパートが存在する
- よくある失敗が記載されている
- 次にやるべきことが記載されている
